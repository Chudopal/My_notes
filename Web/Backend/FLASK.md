### <a name="installing"></a> Установка Flask
+ Установить виртуальную среду: 
    ```bash
    python3 -m venv env
    ```
+ Активировать виртуальную среду:
    ```bash
    source env/bin/activate
    ```
+ Запустить приложение flsk:
    ```bash
    flask start
    ```
